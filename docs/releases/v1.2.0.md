# English patches v1.2.0

This release updates both games with the shared Court Record text hook and
fixes English text layout in interactive tutorials. The release build can now
be recreated from supported Japanese base CIAs, Scarlet Study update CIAs, and
a user-supplied Steam copy of *The Great Ace Attorney Chronicles*.

Install only the two files for the game you own, in update-then-DLC order.

## The Great Ace Attorney: Adventures

Requires the Japanese base game (`000400000014AD00`).

1. `TGAA1-Official-English-v2.8.6.cia` — English update
   (`0004000E0014AD00`, internal version `2.8.6` / 2182).
2. `TGAA1-English-DLC-v1.0.5.cia` — English DLC
   (`0004008C0014AD00`, internal version `1.0.5` / 1029).

Update v2.8.6 adds the native Court Record hook and keeps control tags on the
final page of interactive tutorials, preventing paginated tutorial text from
blocking input. It also validates tutorial height and movie-text width during
every build.

## The Great Ace Attorney 2: Resolve

Requires the Japanese base game (`00040000001AE200`).

1. `DGS2-Official-English-v2.3.3.cia` — English update
   (`0004000E001AE200`, internal version `2.3.3` / 2099).
2. `DGS2-English-DLC-v1.0.1.cia` — translated DLC
   (`0004008C001AE200`, internal version `1.0.1` / 1025).

Update v2.3.3 uses the same shared Court Record hook and applies the tutorial
and movie-layout build guards to TGAA2. The DLC packages are unchanged from
v1.1.0.
