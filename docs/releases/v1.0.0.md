# English patch v1.0.0

First stable release, verified on real Nintendo 3DS hardware.

Requires the Japanese 3DS base game (`00040000001AE200`) and current custom
firmware. Install both assets in this order:

1. `DGS2-Official-English-v2.3.0.cia` — English update
   (`0004000E001AE200`, internal version `2.3.0` / 2096).
2. `DGS2-English-DLC-v1.0.0.cia` — translated DLC
   (`0004008C001AE200`, internal version `1.0.0` / 1024).

The update includes the official English localization for all five main
episodes, restores the DLC entry point, retains the original Capcom startup,
and supports installed DLC without an online entitlement lookup.

The DLC contains direct Japanese-to-English translations of both
3DS-exclusive mini-episodes. Its 2,012 translated entries use the final English
font tracking and hardware-checked line layout. All 42 rebuilt DLC GMD files
pass semantic checks.

The final packages contain the exact same NCCH content archives as the
hardware-tested builds; only their release names and installable title versions
were changed.
