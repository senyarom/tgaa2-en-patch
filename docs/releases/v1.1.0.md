# English patches v1.1.0

This release adds *The Great Ace Attorney: Adventures* and updates *The Great
Ace Attorney 2: Resolve*. Both builds have been tested on Nintendo 3DS
hardware; the final TGAA2 package was also installed and booted in a clean
Azahar setup without a LayeredFS override.

Install only the two files for the game you own, in update-then-DLC order.

## The Great Ace Attorney: Adventures

Requires the Japanese base game (`000400000014AD00`).

1. `TGAA1-Official-English-v2.7.4.cia` — English update
   (`0004000E0014AD00`, internal version `2.7.4` / 2164).
2. `TGAA1-English-DLC-v1.0.3.cia` — English DLC
   (`0004008C0014AD00`, internal version `1.0.3` / 1027).

This is the hardware-tested TGAA1 build with the official English main-game
and extra-content localization, corrected text layout, English system menus,
and a working title-screen DLC entry point.

## The Great Ace Attorney 2: Resolve

Requires the Japanese base game (`00040000001AE200`).

1. `DGS2-Official-English-v2.3.2.cia` — English update
   (`0004000E001AE200`, internal version `2.3.2` / 2098).
2. `DGS2-English-DLC-v1.0.1.cia` — translated DLC
   (`0004008C001AE200`, internal version `1.0.1` / 1025).

Compared with v1.0.0, the TGAA2 update restores a stable three-card title
menu: an unlabeled but functional DLC card, English New Game/Continue, and
English Select Episode. The existing direct Japanese-to-English translation
of both 3DS-exclusive DLC mini-episodes is unchanged.

The DLC card is intentionally left without a text label in both games. Adding
one to Scarlet Study's shared animated texture atlas caused rendering debris;
the blank card preserves correct geometry, highlighting, and input behavior.
